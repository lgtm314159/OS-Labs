1) Change directory to lab4-xin-junyang/lab4 (suppose you are already under
lab4-xin-junyang/):

   cd lab4

>>>>> Please make sure you are in directory lab4-xin-junyang/lab4 before running any of the following commands! <<<<<

2) Compile all the java files by running:

   bash compileLab4.sh

3) Run the program with the required arguments:

   bash runLab4.sh machineSize pageSize processSize jobMix numOfRefsPerProcess algorithm mode
   
   Note: mode should be 0 for no debugging info, 1 for debugging info without random number, and 11 for debugging info with random number.

   The result will be printed out to stdout.
